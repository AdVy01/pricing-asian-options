# pricing-asian-options
This repository showcases 4 different methods to price Asian options:

  1.) Monte Carlo with Control Variate

  2.) Večeř dimension reduction
  
  3.) Laplace Transforn
  
  4.) MPDATA algorithm

It is a supplemental code for [this research task[(insert link) at FNSPE CTU in Prague.

For the example usage, see example_usage.ipynb.
